object-fit

CSS object-fit속성은 컨테이너에 맞게 img태그 또는 video태그 크기를 조정하는 방법을 지정하는 데 사용됩니다.

  • contain : 이미지는 종횡비를 유지하지만 주어진 치수에 맞게 크기가 조정됩니다.

  • cover : 이미지는 종횡비를 유지하고 주어진 치수를 채웁니다. 이미지에 맞게 잘립니다.

  • fill : 기본값입니다. 이미지는 주어진 치수를 채우도록 크기가 조정됩니다. 필요한 경우 이미지에 맞게 늘어나거나 찌그러집니다.

  • none : 이미지의 크기가 조정되지 않습니다.

  • scale-down : 이미지가 가장 작은 버전 none또는 contain
